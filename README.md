# 👨‍💻 Software Engineering Portfolio

A highly interactive, production-ready developer portfolio designed to showcase modern front-end engineering, algorithm visualization, and a focus on quantitative trading algorithms.

## ✨ Advanced Features
- **Live GitHub API Integration:** Asynchronously fetches and displays the most recently updated repositories.
- **Interactive Command Line:** A fully functional, integrated system terminal for recruiter engagement.
- **Live Algorithm Visualizer:** Real-time Bubble Sort DOM manipulation and visualization.
- **Premium UI/UX:** Glassmorphism (frosted glass), 3D card tilt effects, and custom cyber scrollbars.
- **Particle Physics Background:** HTML5 Canvas network nodes that dynamically react to cursor coordinates.
- **Cyber Sound Design:** Web Audio API integration for satisfying micro-interactions.

## 🛠️ Tech Stack
- **Structure:** HTML5 (Semantic & Accessible)
- **Styling:** CSS3 (CSS Variables, Flexbox/Grid, Animations, Media Queries)
- **Logic:** Vanilla JavaScript (ES6+, Async/Await, Intersection Observer)

The project is continuously deployed and hosted for free via GitHub Pages. Simply visit the live URL to interact with the environment.
